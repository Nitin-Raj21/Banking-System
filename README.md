# 🏦 Banking System

A console-based banking system built with Java and MySQL.

## Features
- Create and manage customers
- Open savings and current accounts
- Deposit, withdraw, transfer funds
- Transaction history
- Monthly summary reports

## Tech Stack
- Java (JDBC)
- MySQL
- DAO Pattern

## Database
Run `schema.sql` to set up the database.
